# 🚀 Job Application Tracker

A full-stack web application that helps users track their job applications in one place.  
Built using **Flask**, **MySQL**, **HTML**, **CSS**, and **JavaScript**, with secure authentication and clean UI.

---

## 📌 Features

- 🔐 User Registration & Login
- 🔑 Secure password hashing
- 👤 User-specific job applications
- ➕ Add new job applications
- ✏️ Update application details
- ❌ Delete applications
- 🚪 Logout functionality
- 🌱 Environment variable–based configuration
- 💾 MySQL database integration
